# AI Marketing Automation Agency - Website Specification

本文件為「懂行銷的 AI 實踐家」個人接案網站之開發規格書。本網站旨在協助微型品牌、個人創作者與電商賣家，透過 AI 與自動化技術解決行銷與營運痛點。

## 1. 專案基本資訊 (Project Overview)
*   **網站定位：** 行銷自動化診所 / 輕量級 AI 解決方案展示與接案漏斗。
*   **開發目標：** 建立專業信任感、展示模組化服務、透過內嵌問卷自動過濾高風險客戶，維持低溝通成本的業餘接案模式。
*   **開發者核心優勢 (Agent 需封裝於文案與定位中)：**
    *   **行銷策略規劃：** 具備深厚的市場營運與數位行銷實戰經驗。
    *   **技術實踐能力：** 精通 Python、JavaScript、HTML/CSS。
    *   **專業認證背書：** 持有 IPAS AI 應用規劃師證照。

---

## 2. 網站技術規格 (Technical Stack & Requirements)
*   **前端架構：** HTML5, Tailwind CSS (相容於主流前端框架如 Next.js / Vue 或純靜態網頁)。
*   **響應式設計 (RWD)：** 必須完全相容於 Mobile (iOS/Android) 與 Desktop。
*   **第三方串接：**
    *   表單：內嵌 Google Forms (或自訂 Form 串接 API)。
    *   行事曆：後續保留 Calendly 串接彈性。
*   **合規與安全：** 採用專案交付制，不儲存客戶敏感資料；串接之 API（如 Gemini API）由客戶自行申辦綁定，網站僅負責代碼部署與邏輯串接。

## 4. 頁面區塊文案與詳細規格 (Section-by-Section Spec)

### 4.1 Header (導航列)
*   **規格：** 固定在上方的 Sticky Nav，具備毛玻璃效果 (Backdrop Blur)。
*   **選單項目：** `首頁`、`專業能力`、`服務方案`。
*   **右側按鈕 (CTA)：** `預約健診` (點擊後滑動至底部問卷區)。

### 4.2 Hero Section (首頁開場)
*   **主標題：** 用技術解決行銷難題，把時間留給創意。
*   **副標題：** 結合「行銷策略深度」與「Python/JS 開發實力」，專為個人創作者與微型品牌打造低維護、高回報的 AI 自動化工作流。
*   **行動呼籲按鈕：** `[ 立即填表，預約 AI 效率健診 ]` (錨點跳轉至底部問卷)。

### 4.3 Pain Points (痛點共鳴區)
*   **引入文字：** 身為一人公司或微型團隊，你是否也常面臨這些困境？
*   **四核心痛點卡片設計：**
    1.  **時間被雜務吃掉：** 每天花好幾個小時在處理重複的社群排版、手動複製貼上？
    2.  **數據看得到吃不到：** 電商後台數據一堆，卻沒時間整理成看得懂的報表來調整策略？
    3.  **AI 總是胡說八道：** 嘗試過現成的 AI 客服，卻擔心它回答不精準、虛構產品資訊，反而造成客訴？
    4.  **知道 AI 強大，但不知從何開始：** 每天看著一堆新工具焦慮，卻找不到人幫你真正落地？

### 4.4 Professional Skills (專業能力轉化區)
本區塊需將技術技能轉化為商業價值，避免純資訊列舉：
*   **行銷策略規劃：** 擁有豐富的數位行銷與市場營運實戰經驗，確保 AI 應用聚焦於「營收與成長」。
*   **Python 自動化開發：** 精通 Python 爬蟲與自動化流程，解決現成軟體做不到的客製化效率需求。
*   **RAG AI 知識工程：** 利用 Dify 與 Gemini API，將你零散的品牌文檔，打造為 100% 精準回答的 AI Agent。
*   **專業認證背書：** 持有 IPAS AI 應用規劃師認證，提供系統化、規範化的落地路徑。

### 4.5 Service Packages (四大服務方案)
採卡片式橫向排版或響應式網格，區分技術與服務邊界：

#### 方案一：【AI 策略健診】(策略諮詢/引流服務)
*   **核心價值：** 釐清瓶頸，規劃最適合的 AI 自動化導入路徑圖 (Roadmap)。
*   **技術栈：** 多維度評估表、1-on-1 線上策略對談。
*   **交付物：** 個人/品牌專屬的 AI 自動化落地藍圖 PDF。

#### 方案二：【社群內容轉化引擎】(Python 自動化腳本)
*   **核心價值：** 將核心長文一鍵拆解並自動轉化為各平台（Threads/IG/FB）合適的素材與語氣。
*   **技術規格：** Python 腳本、LLM API 串接、Google Colab 輕量部署環境。
*   **交付物：** 腳本代碼、客製化 Prompt 系統指令、3 分鐘操作教學影片。

#### 方案三：【行銷數據自動化管家】(資料處理/分析)
*   **核心價值：** 自動清洗雜亂後台數據，產出具備行銷洞察的自動化儀表板。
*   **技術規格：** Python (Pandas/Openpyxl) 數據清洗、Google Sheets API 串接或 Looker Studio 視覺化整合。
*   **交付物：** 自動化資料處理腳本、標準化數據儀表板模版。

#### 方案四：【RAG 電商客服 Agent】(旗艦/核心服務)
*   **核心價值：** 建立專屬 AI 店長，嚴格基於你的產品知識庫進行問答，告別 AI 幻覺。
*   **技術規格：** Dify 工作流 (Workflow) 架構、Google Gemini API、知識庫 (Knowledge Base) 分段優化與 Markdown 清洗。
*   **交付物：** Dify 運行環境部署、基礎知識庫清洗（限額頁數內）、AI 語氣與限制詞調教、後台知識更新教學。

## 5. 接案篩選問卷機制 (The Qualification Funnel)
本表單內嵌於網站底部，為客戶進入服務的唯一入口。

### 填寫前提示文字：
> 💡 **填寫須知：**
> 為了保障交付品質，本服務採「專案交付制」，日常溝通以 Email 與專案管理工具為主。請花 2 分鐘填寫以下問卷，我會在評估後主動與您聯繫。

### 問卷欄位結構：
1.  **Q1. 您目前在商務營運上，最感到頭痛、最消耗時間的環節是？ [單選]**
    *   ( ) 社群文案產出與多平台排版發布
    *   ( ) 廣告或電商後台數據雜亂、沒時間分析
    *   ( ) 瑣碎的客戶日常問題回覆
    *   ( ) 我想用 AI 提升效率，但完全沒有頭緒
2.  **Q2. 請用一句話描述，您最希望被「自動化」或交給 AI 處理的任務是什麼？ [簡答]**
3.  **Q3. 針對您想解決的問題，目前手邊的「原始資料」整理狀況如何？ [單選 - 核心過濾題]**
    *   ( ) 已有整理好的產品手冊 / FAQ Word 檔 / 結構化報表
    *   ( ) 只有散落的文件、網站內容或對話紀錄，需要協助整理
    *   ( ) 目前完全沒有任何現成資料
4.  **Q4. 本服務主要針對個人/微型客群提供「輕量化、高效率」的工具部署，您的預期上線時間大約是？ [單選]**
    *   ( ) 希望在一週內快速看到成效
    *   ( ) 一個月內皆可，希望能穩定運行
    *   ( ) 目前還在評估階段，想先諮詢了解
5.  **Q5. 您的聯絡資訊 [必填]**
    *   姓名、Email、Line ID
